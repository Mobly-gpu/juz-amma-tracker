# Hifdh Companion

Upload all files and the icons folder to the root of your GitHub Pages repository. Keep `index.html` in the root. In Settings → Pages, deploy from `main` and `/(root)`.

The app requires HTTPS for installation and microphone recording. Qur’an text is downloaded once and cached. Audio streams from EveryAyah. Progress stays in the browser unless exported or synced to a private GitHub Gist.
